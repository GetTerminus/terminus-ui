API

<ts-tabset>

No Properties

[tsTabHeader]

Properties

tsTabHeader 
string
required
Specifies the ID so the content can be linked to a [tsTabContent]. This must be unique.
isActive 
boolean
false
Sets whether the tab is active. Supports being set to false at which time the closest available tab is activated.
isDisabled 
boolean
false
Sets whether not the tab is disabled. If the tab is active when it is disabled, the closest available tab is activated.

Events

isActiveChange 
boolean
Fires when the tab's active status is changed. Using the [(isActive)] syntax is recommended so that the value is updated when the active tab changes.
activate 
void
Fires when the tab activates.
deactivate 
void
Fires when the tab deactivates.
