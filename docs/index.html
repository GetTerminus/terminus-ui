Adding a Component

  1. Create a directory using the component name: src/lib/src/button/
    • Necessary files:
      • button.module.ts
        • Class name: TsButtonModule
      • button.component.ts
        • Class name: TsButtonComponent
      • button.component.scss
      • button.component.html
  2. Import and export button.component.ts inside button.module.ts
  3. Add TsButtonComponent to the exports and declarations of button.module.ts.
  4. Import TsButtonModule in src/lib/src/module.ts and add it to the TERMINUS_MODULES array.
  5. Export TsButtonModule from src/lib/index.ts.
  6. Comment all methods, constants & @Inputs using the supported JSDoc style.
  7. Add a usage example in the component documentation with every possible input included.
  8. Document styles using nucleus comments.
