terminus-ui/src/chart/options.interface.ts

TsChartAxisType

The type of axis.

In a datetime axis, the numbers are given in milliseconds, and tick marks are placed on appropriate values like full hours or days. In a category axis, the point names of the chart's series are used for categories, if not a categories array is defined.
TsChartAxisType: "category" | "datetime" | "linear" | "logarithmic"
TsChartData

An array of data series'.
TsChartData: TsChartDataItem[]
TsChartLabelFormatterFn

Callback function to format the text of the tooltip from scratch. Return false to disable tooltip for a specific point on series.

Available data: this.percentage|point|series|total|x|y
TsChartLabelFormatterFn: function
TsChartMapTypes

The allowed map types
TsChartMapTypes: "europe" | "northAmerica" | "usa" | "worldContinents" | "world"
TsChartStacking

Define how chart data should stack
TsChartStacking: null | "normal" | "percent"
TsChartVisualizationOptions

Define the supported chart visualizations
TsChartVisualizationOptions: "area" | "bar" | "bullet" | "column" | "line" | "map" | "mapbubble" | "pie" | "scatter"
